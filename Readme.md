# Usage
1. sample.txtのように書く
2. ```python convery.py sample.txt```を実行する
3. sample.txt-output.txtが出力なので確認する


# 記法
- .はインデント(.が一つだとx.x、2つだとx.x.xのようになる)
- -で囲むとその範囲に下線が引かれる
